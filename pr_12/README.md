# Практическая работа 12

Практическая работа 12 оформлена как подготовка к контрольной работе №2 и объединяет результаты практик 7–11.

## Что входит

- `pr_7` — регистрация, вход и CRUD товаров
- `pr_8` — access token и защищённые маршруты
- `pr_9` — refresh token и обновление пары токенов
- `pr_10` — клиентская часть для аутентификации и управления товарами
- `pr_11` — роли, управление пользователями и разграничение доступа

## Проверка

1. Запустить серверные части `pr_7`, `pr_8`, `pr_9`, `pr_10/backend`, `pr_11/backend`.
2. При необходимости запустить клиентские части `pr_10/frontend` и `pr_11/frontend`.
3. Проверить регистрацию, вход, работу access и refresh токенов, а также доступ по ролям.
4. Убедиться, что README и структура репозитория отражают весь комплект работ 7–11.
