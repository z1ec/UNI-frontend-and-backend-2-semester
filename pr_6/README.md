# Практическая работа 6

Практическая работа 6 оформлена как подготовка к контрольной работе №1 и объединяет результаты практик 1–5.

## Что входит

- `pr_1` — карточка товара на HTML и SCSS
- `pr_2` — базовый CRUD API товаров на Express
- `pr_3` — JSON, Postman и тестирование API
- `pr_4` — интернет-магазин на React и Express
- `pr_5` — расширение API документацией Swagger

## Проверка

1. Запустить серверные части `pr_2`, `pr_3`, `pr_4/backend`, `pr_5/backend`.
2. При необходимости запустить клиентские части `pr_4/frontend` и `pr_5/frontend`.
3. Проверить доступность основных маршрутов и интерфейсов.
4. Убедиться, что структура репозитория полная и согласованная.
