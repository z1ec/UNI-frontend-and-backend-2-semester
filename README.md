# UNI-frontend-and-backend-2-semester

Репозиторий содержит полный набор практических работ по дисциплине «Фронтенд и бэкенд разработка» со 1 по 12.

## Структура

- `pr_1` — HTML и SCSS, карточка товара
- `pr_2` — CRUD API товаров на Express
- `pr_3` — JSON, Postman и тестирование API
- `pr_4` — интернет-магазин на React и Express
- `pr_5` — интернет-магазин со Swagger-документацией API
- `pr_6` — подготовка к контрольной работе №1
- `pr_7` — bcrypt-аутентификация и CRUD товаров
- `pr_8` — JWT и защищённые маршруты
- `pr_9` — refresh-токены
- `pr_10` — frontend + backend с хранением токенов на клиенте
- `pr_11` — RBAC, роли и управление пользователями
- `pr_12` — подготовка к контрольной работе №2

## Материалы

В папке `workbooks` находятся исходные PDF-материалы с методическими указаниями для практических работ.
